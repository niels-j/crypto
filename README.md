# crypto
Android app for experimenting with crypto

Installation notes for Android Studio

Firstly, create a new project in Android Studio. You are free to name the project 'crypto'
or using any other project name.

Secondly, in the project, go to the directory <projectname>/app/src/main.
In this directory, you put the sources from GitHub, ie.,
- the file AndroidManifest.XML
- the directory java/nielsj/crypto
- etc.
